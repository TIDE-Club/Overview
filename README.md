# TIDE Club
> 产学研深度融合创新实践平台

## 组织简介
TIDE Club 是依托多所海内外顶尖高校学术资源、联合行业龙头企业打造的跨学科创新实践平台。组织围绕真实行业场景，构建“技术研发-场景验证-产业落地”的全链条协同机制，推动前沿数字技术与实体经济、民生服务深度融合。

核心合作院校与机构包括：香港中文大学（深圳）、北京师范大学-香港浸会大学联合国际学院、清华大学、中国人民大学、约翰霍普金斯大学等。  
核心合作产业方向覆盖：航运物流、商业航天、高端制造、医疗健康、城市规划、数据要素流通等。

## 理念与使命
- **跨学科协同**：连接医学、工业、运筹、数据科学等多学科能力。
- **问题导向研发**：围绕真实行业痛点开展可落地的技术攻关。
- **合规与责任并重**：在医疗、数据等场景严格遵循安全与伦理规范。
- **科研与应用并进**：兼顾论文成果、平台建设与行业应用价值。
