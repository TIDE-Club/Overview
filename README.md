# TIDE Club
> 产学研深度融合创新实践平台

## 组织简介
Tide Club是依托多所海内外顶尖高校学术资源、联合行业龙头企业打造的跨学科、产学研深度融合的创新实践平台。核心合作院校涵盖香港中文大学（深圳）、北京师范大学-香港浸会大学联合国际学院、清华大学、中国人民大学、约翰霍普金斯大学等，合作产业伙伴广泛覆盖航运物流、商业航天、高端制造、医疗健康、城市规划等多个国民经济核心领域，构建了“高校支撑、企业协同、技术落地”的良性生态。

平台以人工智能、大数据、大语言模型、运筹优化、生物信息学等前沿技术为核心底座，聚焦垂直行业真实场景痛点，构建“技术研发-场景验证-产业落地”的全链条创新体系。核心业务布局工业领域、医学领域、运筹领域、数字科技领域四大板块，致力于推动前沿数字技术与实体经济、民生服务的深度融合，为合作单位提供可落地、高价值的数字化解决方案与科研支撑，助力行业数字化转型与高质量发展。

## 理念与使命
- **跨学科协同**：连接医学、工业、运筹、数据科学等多学科能力。
- **问题导向研发**：围绕真实行业痛点开展可落地的技术攻关。
- **合规与责任并重**：在医疗、数据等场景严格遵循安全与伦理规范。
- **科研与应用并进**：兼顾论文成果、平台建设与行业应用价值。
